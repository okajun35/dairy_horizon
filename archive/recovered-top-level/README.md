# 回収したトップ直下ファイル

このディレクトリは、2026-07-14時点でトップ直下に誤配置されていた未追跡ファイルを、現在の実行対象から分離して保管したものです。

- `legacy-python/`: 旧スクリーニング、将来シミュレーション、データ生成に見えるPythonファイル
- `legacy-static/`: 旧表示用JavaScript
- `recovered-data/`: JSON・CSVなどのデモデータ、計算結果、資料メタデータ
- `recovered-docs/`: 内容が仕様書・プロンプト・資料だったファイル

ファイル名と内容が一致しないものがあるため、このアーカイブをアプリから読み込んではいけません。再利用する場合は、内容を検証してから適切なパス・名称で採用します。
