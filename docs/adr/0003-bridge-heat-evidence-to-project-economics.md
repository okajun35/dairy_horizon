# ADR-0003: 暑熱対策の確認値を設備単体の年間便益へ接続する

- Status: Accepted
- Date: 2026-07-17

## Context

牛舎上のカバー頭数と財務試算は存在するが、両者をつなぐ乳量効果が
`3.0kg/頭・日` のデモ仮定に隠れていた。このままでは、暑熱対策と財務が
別の表示に見え、農場実績を反映して結論を更新できない。

## Decision

新規カバー頭数、THI対象日数、防止乳量差から年間防止乳量を求め、実現乳価と
変動費率から年間限界利益効果へ変換する純粋関数を設ける。

ファン台数、カバー頭数、THI対象日数、防止乳量差は別の責務として保持する。
未確認値をLLMや決定論的エンジンが生成しない。

この計算を「設備単体の年間便益」と呼び、農場全体のキャッシュフローとは
呼ばない。年間出荷乳量は増分便益の計算へ混ぜない。

## Consequences

- 暑熱対策の物理的な確認値から、財務結果まで同じ単位で追跡できる。
- 夏季乳量差、実現乳価、電力量単価を後から利用者入力へ置き換えられる。
- 借入、税、農場全体の費用を含まないため、支払能力の断定には使えない。
