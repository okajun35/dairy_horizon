# ADR-0004: 年間出荷乳量は設備採算と分離した売上規模として扱う

- Status: Accepted
- Date: 2026-07-17

## Context

暑熱対策単体の年間差引だけでは、農場全体に対する金額の大きさを読み取りにくい。
一方、搾乳牛頭数に365日や仮の乳期日数を掛けると、妊娠・分娩・乾乳を含む実際の
出荷乳量とは異なる精密そうな値を作ってしまう。

## Decision

現在と5年後の年間出荷乳量を任意の直接入力として受け取り、実現乳価との積を
「売上規模の背景」として表示する。頭数、搾乳日数、現在値から出荷乳量を推定しない。

この売上規模は、暑熱対策単体の年間便益、年間負担、年間差引、回収条件へ渡さない。
将来の年間出荷乳量が未入力なら、将来売上は未評価とする。

## Consequences

- 利用者は対策単体の金額を農場規模と並べて読める。
- 繁殖サイクルを無視した年間乳量をシステムが発明しない。
- 農場全体の費用や資金繰りは未実装なので、売上規模をキャッシュフローや支払能力とは
  表示できない。
