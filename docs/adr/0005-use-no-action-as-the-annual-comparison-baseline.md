# ADR-0005: 追加しない場合を年間比較の基準にする

- Status: Accepted
- Date: 2026-07-17

## Context

設備の電気代と設備費年換算だけを先に示すと、利用者には対策案が単なる
支出に見える。一方、暑熱対策の効果を農場全体の利益と呼ぶと、借入、税、
飼料費などを含まない粗い試算を黒字予測と誤解させる。

## Decision

現在の未カバー推計を基準に、次の三つを同じ年間条件で比較する。

- 追加なしで防げない乳量と乳代売上への影響
- 第1期後に残る損失と追加設備の年間負担
- 頭数目安まで追加した後に残る損失と追加設備の年間負担

案の評価は「何もしない場合との差」として表示する。差は、変動費控除後の
限界利益損失と設備負担を比較した暑熱対策単体の値であり、農場全体の黒字とは
呼ばない。カバー頭数と防止乳量差が仮定の場合は、その出所を併記する。

## Consequences

- 利用者は、支出だけでなく放置した場合の損失と改善幅を比較できる。
- 条件が成立すれば正の改善、成立しなければ負の改善として同じ軸で読める。
- 実測していない乳量効果を利益保証として扱わない。
- 農場全体のキャッシュフロー、借入、税、初期支払時期は別の評価として残る。
